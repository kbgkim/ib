이 코드는 유동화증권을 “발행 형태 + 구조” 기준으로 구분하는 항목입니다.
👉 즉, 어떤 방식으로 투자자에게 자금을 조달했는지를 나타냅니다.

📌 1️⃣ 전체 개념

👉 유동화증권 =
자산(대출, 분양대금 등)을 기반으로 만든 금융상품

👉 종류 구분 기준:

공모 vs 사모
단기 vs 장기
실제 자산 vs 합성 구조

📊 2️⃣ 항목별 상세 설명

🏦 ① ABS (공모)

👉 Asset-Backed Securities (자산유동화증권)

✔ 의미

일반 투자자 대상 공모 발행

✔ 특징

장기 구조
신용평가 필수
규제 엄격

✔ PF에서

안정적 구조
기관투자자 참여

⚡ ② ABCP

👉 Asset-Backed Commercial Paper

✔ 의미

단기(보통 3개월 이하) 유동화증권

✔ 특징

만기 짧음
롤오버(재발행) 필요

✔ PF에서

브릿지 자금
유동성 관리용

👉 핵심 리스크:

👉 롤오버 실패 위험🔥

📄 ③ ABSTB

👉 Asset-Backed Short-Term Bond

✔ 의미

ABCP보다 약간 긴 단기 채권

✔ 특징

단기 + 채권 형태
구조 안정성 조금 더 높음

💼 ④ 사모사채 (사모 ABS)

👉 소수 투자자 대상 유동화 채권

✔ 의미

Private ABS

✔ 특징

구조 유연
맞춤형 설계 가능

✔ PF에서

가장 많이 사용🔥

💳 ⑤ ABL

👉 Asset-Based Loan

✔ 의미

자산을 담보로 한 대출

✔ 특징

증권이 아니라 “대출”
은행 중심 구조

👉 차이:

ABS → 투자상품
ABL → 대출상품

🧪 ⑥ Synthetic-CDO / ABCP

👉 합성 유동화 구조

✔ 의미

실제 자산 이전 없이

👉 신용위험만 분리해서 유동화

✔ 구조

CDS (신용파생상품) 활용

✔ 특징

복잡함
고위험
금융공학 구조

👉 금융위기 때 문제된 구조 🔥

📊 3️⃣ 한눈에 비교

구분	형태	만기	특징
ABS(공모)	증권	중장기	안정
ABCP	증권	단기	롤오버 필요
ABSTB	증권	단기	ABCP보다 안정
사모ABS	증권	다양	유연
ABL	대출	다양	은행 중심
Synthetic	파생	다양	고위험

📌 4️⃣ PF 실무 핵심 🔥

✔ 가장 많이 쓰는 구조

ABCP + 사모ABS 조합

✔ 이유

초기: ABCP (브릿지)
안정화: ABS 전환

👉 구조 흐름:

착공 전 → ABCP
분양 진행 → ABS

📌 5️⃣ 리스크 관점

✔ 위험 순서 (대략)

공모 ABS (낮음)
→ 사모ABS
→ ABSTB
→ ABCP
→ Synthetic (높음)

📌 7️⃣ 한 줄 정리

👉
유동화증권 종류 = “어떤 방식(구조/만기/투자자)으로 자금을 조달했는가”

🚀 핵심 한마디

👉
PF 금융 구조 = “단기 자금(ABCP) → 장기 자금(ABS)로 갈아타는 게임”

원하면 다음 단계도 이어서 설명해드릴게요:

👉 ABCP 롤오버 구조 (실무 핵심🔥)
👉 SPV 구조 + 자금 흐름
👉 실제 PF 딜 구조 전체 그림
