# IPO 공모가 산정 알고리즘 (실전 모델)

## 1. 개요

IPO 공모가 산정은 단순 Valuation이 아니라
다음 4가지 요소를 결합한 **확률적 최적화 문제**이다.

* 기업 가치 (Valuation)
* 수요예측 (Demand)
* 시장 상황 (Market)
* 투자자 행동 (Behavior)

---

## 2. 전체 아키텍처

```
[기업 데이터]
    ↓
[Valuation (Anchor Price)]
    ↓
[수요예측 모델]
    ↓
[시장 보정]
    ↓
[최적 공모가 결정]
```

---

## 3. Step 1 — Anchor Price (기초 가치)

### 3.1 개념

기업의 이론적 가격 범위 산정

### 3.2 수식

```
Anchor Price = w1 × PER 기반 + w2 × EV/EBITDA 기반
```

---

### 3.3 상세 계산

```
PER 기반 가격 =
    (Peer PER 평균 × 예상 순이익) / 총 주식수

EV/EBITDA 기반 가격 =
    ((Peer EV/EBITDA × EBITDA) - Net Debt) / 총 주식수
```

---

### 3.4 결과

```
예: 18,000 ~ 24,000원
```

---

## 4. Step 2 — 수요예측 모델 (핵심)

### 4.1 목적

투자자 주문 데이터를 기반으로 가격별 수요 계산

---

### 4.2 수요곡선 정의

```
Q(p) = Σ (qi * 1(pi ≥ p))
```

* p: 공모가 후보
* qi: 주문 수량
* pi: 주문 가격

---

### 4.3 핵심 지표

#### (1) 경쟁률

```
Demand Ratio = 총 주문량 / 공모 물량
```

#### (2) 고가 지지율

```
High Price Support Ratio =
    (상단 가격 이상 주문량) / 총 주문량
```

---

### 4.4 주요 Feature

* 기관 비중
* 외국인 비중
* 앵커 투자자 참여 여부
* 주문 집중도 (HHI)
* 가격별 수요 분포

---

## 5. Step 3 — 시장 보정

### 5.1 목적

시장 상황 반영하여 가격 조정

---

### 5.2 보정 수식

```
Padj = Panchor × (1 + αM + βV + γS)
```

---

### 5.3 변수 정의

| 변수 | 의미     |
| -- | ------ |
| M  | 시장 수익률 |
| V  | 변동성    |
| S  | 투자 심리  |

---

### 5.4 해석

* 시장 상승 → 가격 상승
* 변동성 증가 → 가격 하락
* 투자 심리 개선 → 가격 상승

---

## 6. Step 4 — 최적 공모가 결정

### 6.1 목적 함수

```
max p * Q(p) - λ * Risk(p)
```

---

### 6.2 Risk 정의

```
Risk(p) =
    수요 부족 확률
  + 상장 후 하락 확률
```

---

### 6.3 해결 방법

* Grid Search
* Gradient Optimization

---

## 7. ML 기반 확장 모델

### 7.1 입력 데이터

### 7.2 출력


### 7.3 주요 Feature

## 8. 실제 의사결정 프로세스

```
1. Anchor Price 산출
2. 수요예측 반영
3. 시장 보정 적용
4. ML 모델로 수익률 예측
5. Risk 고려 최종 가격 선택
```

---

## 9. 핵심 인사이트

* 공모가는 "기업 가치"로 결정되지 않는다
* 수요 + 심리 + 타이밍이 핵심
* 투자자 행동 데이터가 가장 중요

---

## 10. 한 줄 요약

IPO 공모가는
**Valuation 기반 가격을 시작으로 수요와 시장을 반영해 최적화하는 문제**이다.
