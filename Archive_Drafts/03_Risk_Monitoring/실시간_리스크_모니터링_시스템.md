# 실시간 PF 리스크 모니터링 시스템 (Real-Time PF Risk Monitoring System)

---

# 1. 개요

## 목적
PF(Project Finance) 프로젝트의 리스크를 실시간으로 감지하고  
👉 **부도 위험을 조기 탐지하여 대응**

---

## 핵심 기능

- 실시간 데이터 수집
- 리스크 지표 자동 계산 (DSCR, LTV 등)
- ML 기반 부도 확률 산출
- 위험 알림 (Alert)
- 대시보드 시각화

---

## 전체 구조


[데이터 수집]

[스트리밍 처리 (Kafka)]
↓
[리스크 계산 엔진]
↓
[ML 모델 (LightGBM)]
↓
[리스크 DB 저장]
↓
[API 서버]
↓
[대시보드 / 알림]

from kafka import KafkaConsumer
import json

consumer = KafkaConsumer(
    'pf.sales',
    bootstrap_servers='localhost:9092',
    value_deserializer=lambda m: json.loads(m.decode('utf-8'))
)

for message in consumer:
    data = message.value
    print(data)
4. 리스크 계산 엔진
핵심 계산 지표
DSCR
DSCR = Cash Flow / Debt Service
LTV
LTV = Loan / Asset Value
추가 파생 변수
def calculate_features(data):
    data["dscr_buffer"] = data["dscr"] - 1.0
    data["sales_velocity"] = data["sales_rate"] / data["progress_rate"]
    data["risk_score"] = data["ltv"] * (1 - data["sales_rate"])
    return data
5. ML 모델 연동 (LightGBM)
모델 로딩
import lightgbm as lgb

model = lgb.Booster(model_file="pf_model.txt")
실시간 예측
def predict_risk(features):
    prob = model.predict([features])[0]
    return prob
6. Risk Scoring
def map_risk(prob):
    if prob < 0.1:
        return "AAA"
    elif prob < 0.2:
        return "AA"
    elif prob < 0.3:
        return "A"
    elif prob < 0.4:
        return "BBB"
    else:
        return "High Risk"

엔드포인트

GET /pf/risk/{projectId}
Controller 예시
@RestController
@RequestMapping("/pf/risk")
public class RiskController {

    @PostMapping("/evaluate")
    public RiskResponse evaluate(@RequestBody RiskRequest req) {
        double prob = riskService.predict(req);
        String grade = riskService.map(prob);

        return new RiskResponse(prob, grade);
    }
}
